# spring-rest-hero
Spring Boot REST Hero application example

## Get

find all
curl -i http://localhost:8080/api/v1/heroes

get one
curl -i http://localhost:8080/api/v1/heroes/1

## Post
    curl -i -X POST -H "Accept:application" http://localhost:8080/api/v1/heroes
